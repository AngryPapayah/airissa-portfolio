# Portfolio Airissa

Een persoonlijke portfolio website om mijn vaardigheden en projecten te showcasen, gebouwd met React en Vite.

## 🛠️ Technologieën

- [React](https://react.dev/) — UI framework
- [Vite](https://vitejs.dev/) — Build tool met Hot Module Replacement (HMR)
- ESLint — Code kwaliteit
